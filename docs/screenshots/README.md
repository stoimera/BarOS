# Screenshots

Screenshots are served from `public/screenshots` and embedded in `README.md`.

## Folder structure

- `public/screenshots/customer-perspective/`
- `public/screenshots/admin-perspective/`

## Naming guidance

- Use descriptive names (for example `Bookings.png`, `Menu-management.png`).
- Keep names stable once linked from docs.
- Prefer PNG/WebP with readable text at common laptop viewport widths.

## Current usage

The root `README.md` embeds images with paths like `./public/screenshots/...` so they render on GitHub; at runtime the same files are available under `/screenshots/...`.
